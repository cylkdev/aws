# AWS

Elixir library wrapping AWS services (S3, EventBridge, CloudWatch, IAM, Identity Center, Organizations)
with consistent error handling, response deserialization, and sandbox support for testing.

## Services

- `AWS.S3` — object storage
- `AWS.EventBridge` — event routing
- `AWS.CloudWatch` — metrics and alarms
- `AWS.IAM` — identity and access management
- `AWS.IdentityCenter` — IAM Identity Center (SSO)
- `AWS.Organizations` — AWS Organizations
